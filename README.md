# 📊 Customer Churn Prediction App

## 🔍 Problem Statement

Businesses often struggle to identify customers who are likely to stop using their services. Without early detection, companies lose revenue and customer retention becomes difficult.

---

## 🎯 Objective

To build a machine learning application that:

* Analyzes customer behavior
* Predicts whether a customer will churn
* Helps businesses take proactive decisions

---

## 💡 Solution

This project uses machine learning models to predict customer churn based on features like:

* Age
* Tenure
* Usage Frequency
* Support Calls
* Payment Delay
* Subscription Type
* Contract Length
* Total Spend

The app is built using Streamlit for an interactive user interface.

---

## 🛠️ Tools & Technologies

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn

---

## ⚙️ Features

* 📊 Data Overview
* 🔍 Exploratory Data Analysis (EDA)
* ⚙️ Data Preprocessing
* 🤖 Model Training (Random Forest, Logistic Regression, SVM)
* 🔮 Batch Predictions (Upload CSV)
* 👤 Single Customer Prediction

## 🚀 How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/MohammadAta-BIET7522/customer-churn-prediction.git
cd your-repo-name
```

### 2. Create virtual environment

```bash
python -m venv myenv
myenv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the app

```bash
python -m streamlit run app.py
```
